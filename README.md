# Getting Started

## Dependencies

- yarn,
- [Node.js 12.22.0](https://nodejs.org/en/) or later

First, run the development server:

```bash
run yarn
run yarn dev
run yarn api (new tap)
```

Open [http://localhost:3333](http://localhost:3333) with your browser to see the result.
